# Getting Started

## Requirements

- Node.js.
- pnpm.
- Rust toolchain required by Tauri.

## Installation

From the repository root:

```bash
pnpm install
```

## Running Locally

The package scripts define:

```bash
pnpm dev
pnpm tauri
```

## Verification Scripts

The package scripts define:

```bash
pnpm lint
pnpm typecheck
pnpm test
pnpm build
```

## Notes

- This documentation pass did not run install, dev, build, test, lint, or typecheck commands.
- Local screenplay files and project folders should be treated as user data.
